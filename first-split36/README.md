# First-split36
36 key split column staggered keyboard. Use Pro Micro microcontroller and crkbd firmware. Mirror boards.

![](./img/20230120_204041.jpg)

## Parts
- 2x Arduino Pro Micro (Atmega32U4)
- 2x trrs connectors
- 1x 3,5mm audio cabel
- 36x cherry mx switches
- 36x d3 smd diods
